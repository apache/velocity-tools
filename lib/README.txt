R E A D M E

This directory is the central location for shared class 
libraries of all subprojects under jakarta-velocity-tools. 
Following is an overview of which subprojects depend on 
which class libraries.

When adding or updating libraries here please make sure
that the file names include the library version.


VelocityStruts:
- commons-validator.jar (1.0.2)
- jdbc2_0-stdext.jar (for jdk 1.3)
- servlet.jar (2.2+)
- sslext.jar
- struts.jar (1.1)
- velocity-dep-1.3.1.jar

VelocityView:
